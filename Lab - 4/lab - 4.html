¿Qué diferencias y semejanzas hay entre Java y JavaScript?
Java crea aplicaciones que se ejecutan en una máquina o explorador virtual, mientras que el código JavaScript sólo se ejecuta en un explorador. El código Java necesita compilación, mientras que el código JavaScript está en todo el texto.
¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
setFullYear(), setMonth(), setDate(), setHours(), setMinutes(), setSeconds(), setMilliseconds, setTime(), según el valor que queramos modificar.
¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
Métodos de arreglo JavaScript: unshift() - Inserta un elemento al inicio del arreglo. pop() - Remueve un elemento del final del arreglo. shift() - Remueve un elemento del principio del arreglo. slice() - Crea una copia sombra del arreglo.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
El uso indiscriminado de variables globales produce efectos colaterales. Esto sucede cuando existe una alteración no deseada del contenido de una variable global dentro de una función, bien por invocación, bien por olvidar definir en la función una variable local o un parámetro formal con ese nombre.
